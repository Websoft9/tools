== Description ==

本插件是给网页的一级目录增加增加一个qq在线客服窗开，窗开采用固定式

= Key features: =
* 设置3个qq号码
* 设置电话号码
* 设置链接

= Add-on for the plugin: =
* 暂时没有到WordPress的插件市场上上架

= Documents and Support: =
暂无

= Languages supported: =
* Chinese (zh_CN) - by OSHOP



== Installation ==

1. 直接将插件上传到Plugins文件夹
2. 通过WordPress的'插件管理'功能启用插件
3. 暂时没有设置功能.
