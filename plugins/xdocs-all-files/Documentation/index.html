Plugin INSTALLATIONS GUIDE

If You are using multisite wordPress then visit Network Admin area to install plugins.

There are two methods to install plugins in WordPress. You can choose any method you like. If you are getting any error while uploading plugin please use second method

 

Installing from WordPress Dashboard

In WordPress 2.7 and above you can install plugins directly from the admin area. Download the plugin to your computer, then log in to your WP admin area and go to Plugins > Add New. Browse to the plugin archive and select it. Then click Install Now and the plugin will be installed shortly.

 

Installing Plugin Via FTP

To install a plugin you generally just need to put the plugin file into your ‘wp-content/plugins’ directory. Once a plugin is installed, you may activate it or deactivate it from the Plugins menu in your WP administration.

Creating Documents

After installing and activating XDocs now you will able to see XDocs in you wordpress dashboard menu. XDocs is using custom post type same as default posts to create documents but with some advance options.

Add New Post

  • Click the XDocs tab.
  • Click the Add New Sub Tab.
  • Start filling in the blanks. You can Use our Shortcodes!
  • As needed, select a category and make other selections from the sections below the post. Each of these sections is explained below.
  • When you are ready, click Publish.

Sections

Each document can have multiple sections. New Sections can be added to post by clicking at “Add New Section”

Introduction Section

Introduction section use default title, content editor for short summary and featured image field for item image.

Steps

You can also create Steps or subsections for any sections subsection title will be visible in main sidebar menu of this theme.

Frequently Asked (FAQ's)

Can I create multiple documents?

Yes, You can create as many documents as you want just like simple wordpress post.

Can I download documents?

Yes, Click at download zip button and XDocs will generate a zip with all assets and post content.

Do I have to purchase license for each downloaded zip?

No, You have to purchase XDocs only once and you can create as many docs as you want. But If you are planing to install Xdocs on another WordPress then please purchase new license.

XDocs Themes

This is default theme. There 2 more simple themes available. You can select your desire theme. Theme2 does not support steps and theme 3 does not support floating steps nav because of design requirements.

Xdocs theme default theme also have some color varitions.

Preloader Settings

XDocs also have preloader option with some simple flat loading gifs. You can enable or disable preloader if you want.
